FAQ

AIガバナンス成熟度評価とは何ですか？
+
AIガバナンス成熟度評価は、企業のAI利活用におけるガバナンス体制の成熟度を5段階で評価する仕組みです。METI（経済産業省）のAI事業者ガイドライン（第1.1版）の10原則・28要件への対応度を測定し、ISO/IEC 42001:2023（AIMS）認証準備に活用できます。JPGovAIは、この評価プロセスと認証準備を効率化するSaaSです。
コンサルなしでISO 42001認証は取れますか？
+
はい、Enterpriseプランをご利用いただくことで、専門コンサルなしでの認証取得を支援します。JPGovAIは27の要求事項に対応した証拠テンプレート・チェックリスト・改善アクションを提供します。ただし、最終的な認証審査は認定機関（JAB等）が行います。複雑な組織構造をお持ちの場合は、当社の専任サポート担当にご相談ください。
ISO 42001認証とMETIガイドラインの違いは何ですか？
+
ISO 42001はAIマネジメントシステムの国際規格で、第三者認証機関による審査・認証が伴います。METIガイドラインは日本国内向けのソフトロー（法的拘束力なし）で、自己適合宣言形式です。AI推進法は法律（2025年6月施行）で、一定条件下では法的義務が生じます。JPGovAIは3つすべてに対応し、重複する要件をまとめて効率よく対応できます。
データのセキュリティは安全ですか？
+
JPGovAIはISO/IEC 27001準拠のセキュリティ基準を採用しています。データは日本国内のデータセンターに保存され、通信はすべてTLS 1.3で暗号化されます。アップロードされたエビデンス書類はSHA-256でハッシュ化・改竄検知が行われます。また、当社自身もJPGovAIを使ってAI推進法・METIガイドラインへの対応を実施しています（自社適用実績）。
