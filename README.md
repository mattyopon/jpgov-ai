# JPGovAI

**ISO 42001認証準備 & AIガバナンス管理 SaaS**

AI事業者ガイドライン（第1.1版）・ISO/IEC 42001・AI推進法の3規制に対応した、日本初のAIガバナンス管理SaaS。

## 対応規制

| 規制 | 内容 | 要件数 |
|------|------|--------|
| **METI AI事業者ガイドライン v1.1** | 経済産業省・総務省（2025年3月） | 10原則・28要件 |
| **ISO/IEC 42001:2023** | AIマネジメントシステム国際標準 | 7条項・25要求事項 |
| **AI推進法** | 2025年6月施行 | 6章・15要件 |

## 特徴

- 25問の質問票で30分でAIガバナンス成熟度を可視化
- 3規制のクロスマッピングで一元管理
- AI（Claude API）による改善アクション自動提案
- SHA-256ハッシュチェーン + Merkle Tree による改竄防止監査証跡
- ポリシーテンプレート5種の自動生成（AI利用ポリシー、リスク管理方針、AI倫理方針、データ管理方針、インシデント対応手順書）
- PDF レポート出力、CSV/JSON エクスポート

## クイックスタート

```bash
# 依存関係インストール
pip install -r requirements.txt

# 環境変数設定
cp .env.example .env
# .env に ANTHROPIC_API_KEY を設定（省略時はフォールバック提案を使用）

# APIサーバー起動
uvicorn app.main:app --reload
```

ブラウザで http://localhost:8000/docs にアクセスするとAPI仕様を確認できます。

Streamlit UIを使う場合:

```bash
streamlit run ui/streamlit_app.py
```
